# ITEMS — Phase P6 Autonomous Core Completion and Cross-Project ComfyUI Bridge (doc 24)

Goal: close a human-free autonomous MaskFactory runtime, preserve optional independent-accuracy and
scale/DAZ claims as separate profiles, and connect MaskFactory to the main ComfyUI controller through a
versioned, authority-safe producer/consumer bridge. Every item is initially open; implementation or
planning text alone is not completion evidence.
